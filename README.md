## За проекта

**Mendeleev Lab** е уеб приложение, което помага на ученици да изучават химия по интерактивен начин. Включва периодична таблица с информация за всеки елемент, 3D визуализации на молекули, упражнения за изравняване на химични уравнения и викторина.

---

## Функционалности

### 🏠 Начална страница
- Биография и факти за Дмитрий Менделеев
- Обяснение на Периодичния закон
- Закономерности в периодичната система (периоди, групи)
- Основни категории елементи
- Хронология на ключови открития

### 📊 Периодична таблица
- Всичките 118 химични елемента
- Филтриране по категория (алкални метали, халогени, инертни газове и др.)
- Търсене по име или символ
- Кликване върху елемент за подробна информация
- Линк към Wikipedia статия (отваря се в popup прозорец)
- Хоризонтален скролинг на мобилни устройства

### 🔬 3D Молекули
- Интерактивна 3D визуализация на молекули (Three.js)
- Въртене чрез мишка/touch
- Информация и факти за всяка молекула
- Легенда с цветовете на атомите

### ⚗️ Лаборатория
- **Обучение** — стъпки за изравняване на химични уравнения (accordion)
- **Игра за изравняване** — три нива на трудност (Лесно / Средно / Трудно)
- **3D модел** на молекули за текущото уравнение
- **Викторина** с въпроси от химия
- Резултати и статистика

### 🌙 Тъмна тема
- Превключване между светла и тъмна тема
- Запазване на избора в `localStorage`

## Структура на файловете

```
MendeleevLab/
│
├── index.html              # Начална страница (Менделеев и периодичен закон)
├── periodic-table.html     # Периодична таблица
├── molecules.html          # 3D Молекули
├── laboratory.html         # Лаборатория (игра, викторина, обучение)
│
├── styles.css              # Всички стилове (light + dark theme)
├── script.js               # Обща логика (навигация, периодична таблица, викторина)
├── molecules.js            # Three.js логика за 3D молекули
├── balanceGame.js          # Логика на играта за изравняване
├── balanceMolecule3d.js    # Three.js 3D молекула в играта
├── animations.js           # Scroll анимации и активен nav link
│
└── img/
    ├── mendeleev.png           # Портрет на Менделеев
    └── npmg-logo-768x790.png   # Лого на НПМГ (също favicon)
```

## Технологии

| Технология | Употреба |
|---|---|
| HTML5 | Структура на страниците |
| CSS3 | Стилове, анимации, dark theme, responsive design |
| JavaScript (ES6+) | Интерактивност, логика на игра |
| [Three.js](https://threejs.org/) r160 | 3D визуализация на молекули |
| Google Fonts — Inter/Poppins | Типография |
| Google Maps Embed | Карта в footer-а |

> Няма npm пакети, билд система или frameworks — проектът работи директно в браузър.

## Как да стартирате

Просто отворете `index.html` в браузър. Не е нужен сървър.

```
Отвори → index.html
```

> **Забележка:** Three.js се зарежда от CDN, затова е нужна интернет връзка за 3D молекулите.
